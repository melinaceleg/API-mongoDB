# Node.js - MongoDB UNMDP Project

## Run Locally

Clone the project

Go to the project directory

```bash
  cd API-mongoDB
```

Install dependencies

```bash
  npm install
```

Create an ENV file and provide environment variables

```bash
  .env
```

Start the server

```bash
  npm run start
```
